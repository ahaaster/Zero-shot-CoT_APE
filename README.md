## Pipenv
### Installation
```
pip install pipenv
```

#### Use
Installing packages for the virtual environment
```
pipenv install [package]
```

You spawn the virutal environment with the following terminal command
```
pipenv shell
```
Running a python file (while in virtual environment)
```
python [filename].py
```
