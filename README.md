## Pipenv
### Installation
```
pip install pipenv
```

#### Use
In de terminal start je de virtual environment als volgt:
```
pipenv shell
```

Installing packages within for the virtual environment
```
pipenv install [package]
```
Running a python file (while in virtual environment)
```
python [filename].py
```
